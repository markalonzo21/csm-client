# CSM Report Application Frontend

## Tech Stack

- NuxtJS
- SocketIO
- TailwindCSS
- BootstrapCSS
- Sass

## Development Setup

```bash
# install dependencies
$ npm install
# serve with hot reload at localhost:3000
$ npm run dev
```
